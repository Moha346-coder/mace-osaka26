## 📘 What is mace-osaka26?

Mace-osaka26 provides a set of machine learning interatomic potential (MLIP) models designed to work across many different types of materials. It is an upgrade to the earlier mace-osaka24, adding data for heavy elements including many rare and actinide elements. This lets it handle a broader range of chemical substances.

The software is intended for users who want reliable models to predict and simulate the behavior of atoms in crystals, molecules, and heavy-element materials. It supports research in fields like energy resource development and materials design, especially where unusual or complex elements are involved.

## 📄 About the Models

- **MACE-Osaka24:**  
  Covers crystals and molecules. Uses a technique called *Total Energy Alignment* to combine different types of data for better accuracy.

- **MACE-Osaka26:**  
  Builds on MACE-Osaka24. Adds heavy elements from the *HE26* dataset. Covers 97 elements, including minor actinides.  
  This lets the model work well for materials used in advanced energy applications.
